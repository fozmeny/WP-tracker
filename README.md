# WhatsApp Online Tracker

This project tracks when a specific contact on WhatsApp is online using Selenium and Python. The project is set up and run through Anaconda.

## Prerequisites

- Python 3.x (recommended to use Anaconda)
- Selenium
- Google Chrome
- ChromeDriver
